这是一场关于规律作息、坚持运动、认真生活的残酷共学。

## 学习资料/课程安排

每日自律小目标

1. 作息规律
早上 8:00 前起床，不拖延、不赖床，迎接清爽的一天。
晚上 23:30 前入睡，好好休息，为第二天蓄力。

2. 坚持运动
每天至少运动一次，时间不限、形式不限。
不必追求强度，动起来，就是今天的胜利。

3. 自我要求
好好生活，好好爱自己，
用稳定的节奏，慢慢变成更喜欢的样子。
